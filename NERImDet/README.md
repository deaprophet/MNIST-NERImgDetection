# Animal NER + Detection Pipeline

NER + YOLO-based pipeline for verifying animal mentions in text against images.

## Dataset

### NER Dataset

- **Size:** ~10,000 samples
- **Format:** CSV (text + character-level annotations)
- **Tagging:** BIO scheme (`B-ANIMAL`, `I-ANIMAL`, `O`)
- **Source:** Generated by LLM

### Detection Dataset

- **Classes:** 80 animal categories
- **Format:** Class-folder structure with `Label/` subfolder per class
- **Source:** [Kaggle](https://www.kaggle.com/datasets/antoreepjana/animals-detection-images-dataset?resource=download-directory&select=train)

## Models

- **NER:** RoBERTa-base fine-tuned for token classification
- **Detection:** YOLOv8n fine-tuned on animal detection dataset

## Usage

### EDA

See `eda.ipynb` for the complete dataset creation process

### Demonstration

See `demo.ipynb` for the demo testing of pipeline

### Training

#### NER

```bash
python train.py --dataset_path ../data/ner/ner_dataset.csv --output_dir ./checkpoints/ner_model --epochs 3 --batch_size 32 --learning_rate 1e-5
```

#### Image Detection

```bash
python gr.py --train_path "./data/train" --test_path "./data/test" --output_dir "./checkpoints/yolo" --base_model "yolov8n.pt" --epochs 50 --batch_size 16
```

### Inference

#### NER

```bash
python inference.py --model_path ./checkpoints/ner_model --text "Is there a goat on that picture?"
```

#### Image Detection

```bash
python inference.py --model_path "./runs/detect/checkpoints/yolo/weights/weights/best.pt" --image "path/to/img"
```

### Pipeline Inference

```bash
python pipeline.py --ner_model "./NER/checkpoints/ner_model" --yolo_model "./ImDet/runs/detect/checkpoints/yolo/weights/weights/best.pt" --text "Is there a bear on the screen?" --image "path/to/img"
```
